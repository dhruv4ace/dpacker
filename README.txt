This tool pack all the UV's or all the different shapes on any 3D model into a 2D plan, This tool majourly use in Game Development and 3D printing.
The engine file of my tool is present inside win>dhpm
my engine use and AI and algorithm to analysis the shapes and find a perfect solution in which he can fit all of them on a 2D plain. i had use python defualt libaries in this.

This tool is only works in blender , because blender is an open source software.

***WARNING CHANGE THE FOLDER NAME TO 'dpacker' otherwise tool wont work, first extract this folder then change the name of folder to 'dpacker' , then make it zip and run it
